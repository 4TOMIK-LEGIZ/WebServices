# 4TOMIK-PROJECT

## Installation

$ npm install

## Running the app

```
$ npm install
$ npm run build
$ npm run start:dev
```

## Test

```
$ npm run test
$ npm run test:e2e
$ npm run test:cov
```

## Environment variables

```
MYSQL_TOMIK_PROJECT=mysql://{user}:{password}@{host}:{port}/{database}
MYSQL_TOMIK_PROJECT=mysql://root:{...}@localhost:3306/tomik
```
